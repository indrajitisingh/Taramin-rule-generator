## 📌 Overview
A Streamlit-based tool that converts protocol steps into Tamarin Prover rules automatically, simplifying security protocol modeling.

## 🚀 Features
- Convert protocol text into formal rules
- Supports multi-step communication
- Generate `.spthy` rules
- Clean UI using Streamlit

## 🧠 Example Input
User -> Server: Login  
Server -> User: Token  

## 🛠 Tech Stack
- Python
- Streamlit


## ▶️ How to Run
pip install streamlit
streamlit run app.py

## ⚙️ How It Works

1. Enter protocol steps  
2. App processes input  
3. Converts into Tamarin rules  
4. Displays output  
5. Generate `.spthy` rules

## 🎯 Use Cases

- Security protocol analysis  
- Academic projects  
- Learning Tamarin Prover
